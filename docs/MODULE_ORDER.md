# Module Order Reference

This document lists the logical platform module areas.

The order is informational only and does not enforce runtime dependencies.

Current modules:

1. Core Platform
   - authentication
   - org isolation
   - environment isolation

2. Workflow Engine
   - workflow definitions
   - execution engine
   - dry run engine
   - step handler registry

3. Governance
   - approval system
   - policy engine

4. Integrations
   - connectors
   - connector secrets
   - rate limiting

5. Data Processing
   - ingestion pipeline
   - background workers
   - metrics and rollups

6. Frontend
   - Next.js UI
   - API proxy routes
   - platform dashboards

This document exists only as a documentation reference for architecture prompts.
